# MiDrive Server

MiDrive Server is a lightweight personal file server that lets you access, upload, download, and manage files from any device using a web browser.

The server runs locally on your computer and can create a public internet link using ngrok, allowing remote access to your files.

## Features

* Web-based file explorer
* Upload and download files
* Create folders
* Trash system with restore option
* Login system with username and password
* Remote access through a public link
* Real-time connected users counter
* Server control panel with graphical interface
* Change username and password from the application

## How It Works

MiDrive Server starts a local web server on your computer.
You can access it locally through your browser or share it with others using a public link generated with ngrok.

## Installation

1. Download the project files or clone the repository.

2. Install Python 3.10 or newer.

3. Install the required Python libraries.

pip install flask requests customtkinter

## Configuring ngrok

To allow internet access you need to connect ngrok to your account.

1. Create an account

https://dashboard.ngrok.com/signup

2. Get your Authtoken

https://dashboard.ngrok.com/get-started/your-authtoken

3. Run this command in a terminal:

ngrok config add-authtoken YOUR_TOKEN_HERE

Example:

ngrok config add-authtoken 2ABCD123456789xxxxxxxxxxxxx

This only needs to be done once.

## Running the Server

Start the server with Python:

python Server.py

The application will open the control panel where you can:

* start the server
* generate a public link
* copy the link
* view connected users
* see server logs
* change username and password

## Accessing the Drive

Local access:

http://localhost:5000

Public access:

Use the link generated by ngrok.
